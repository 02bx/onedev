Commit Message Transform
This setting tells OneDev how to transform commit message for displaying. Search For expects a Java pattern to match in the commit message, for instance, you may match JIRA issue key with ([A-Z]{2,}-\d+), and specify Replace With as <a href="http://track.example.com/browse/$1">$1</a> to replace JIRA issue keys with issue links. Here $n represents nth capturing group in the pattern
