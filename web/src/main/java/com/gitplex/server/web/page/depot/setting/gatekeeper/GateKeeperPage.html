Add Gatekeeper
  • Commits have to pass all Gatekeepers defined here in order to be accepted by the repository
  • If a gatekeeper is dragged and dropped on another gatekeeper, their positions will be swapped
  • If a gatekeeper is dragged and dropped on Add/Define placeholder, it will be moved to the placeholder position
  • A gatekeeper can not be dragged to a position containing itself or contained by itself
  • Check out examples here to get better understanding of gatekeepers
