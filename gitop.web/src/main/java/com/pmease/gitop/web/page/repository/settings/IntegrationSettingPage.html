Integration Setting

Define strategies to integrate pull requests sent to this repository. Target and source branch of the pull request will be used to find first matching strategy defined below. If no matching strategy can be found, the default strategy will apply.

Strategies

Branch patterns specified in above table should follow the pattern set format. When match against branch patterns, target branch will use the simple branch name, while source branch will use its full name in form of:
<user name>/<repository name>:<branch name>.

Some examples to match target branch:

**
match all target branches
release/**
match all target branches beginning with release/

Some examples to match source branch:

**
match all source branches
robin/*:release
match all source branches with name release from all repositories of user robin
**:release/**
match all source branches begining with name release/ from all repositories
Default Strategy

Default strategy will be used if target and source branch of pull request can not match any of strategies defined above.
